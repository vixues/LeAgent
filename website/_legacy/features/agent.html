如何上手

三步进入可用智能体

  1. 1

    接入模型与租户

    配置 LLM 密钥与 tier 路由；启用 JWT / API Key 与 workspace（参见企业页）。

  2. 2

    装配 Prompt 与工具

    选择模板变体，挂载工具注册表与 Skills；按需打开 MCP。

  3. 3

    会话内迭代

    用户消息驱动循环直至 Terminal；工具结果写回 SessionManager，可选 GenUI 输出。
