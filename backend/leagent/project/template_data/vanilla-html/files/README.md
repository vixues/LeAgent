# Vanilla HTML scaffold

A zero-dependency static site. The supervisor boots it with
`python -m http.server` so it works on every machine without npm.

## Files

- `index.html` — markup.
- `styles.css` — styling.
- `app.js` — behaviour.

The coding agent can extend this scaffold by editing these files
or by adding new ones. To test changes, restart the dev server
from the UI or call `coding_project_run` again.
